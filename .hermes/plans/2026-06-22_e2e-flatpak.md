## Phase 1: End-to-End-Test

### Aufgabe 1: Integrationstest — DB → Dienst → IMAP (Mock)

**Ziel:** Ein Test, der den gesamten Flow durchläuft: Konto anlegen → IMAP verbinden (Fake) → Ordner laden → Nachrichten abrufen → im Cache prüfen.

**Dateien:**
- Erstellen: `pruefungen/tst_integration.cpp`
- Ändern: `pruefungen/CMakeLists.txt`

**Test-Code:**

```cpp
#include <QtCore>
#include <QtTest>
#include "speicher/datenbank.h"
#include "speicher/zwischenspeicher.h"
#include "dienst/konto_dienst.h"
#include "dienst/postfach_dienst.h"
#include "protokoll/imap_verbindung.h"

using MailAdler::Speicher::Datenbank;
using MailAdler::Speicher::Zwischenspeicher;
using MailAdler::Dienst::KontoDienst;
using MailAdler::Dienst::PostfachDienst;
using MailAdler::Protokoll::ImapVerbindung;

class FakeSocket : public QTcpSocket {
    Q_OBJECT
public:
    void simuliereEmpfang(const QByteArray &daten) {
        m_puffer.append(daten);
        emit readyRead();
    }
    qint64 readData(char *data, qint64 maxlen) override {
        qint64 len = qMin(maxlen, (qint64)m_puffer.size());
        memcpy(data, m_puffer.constData(), len);
        m_puffer.remove(0, len);
        return len;
    }
    qint64 bytesAvailable() const override {
        return m_puffer.size() + QTcpSocket::bytesAvailable();
    }
private:
    QByteArray m_puffer;
};

class TestIntegration : public QObject {
    Q_OBJECT

private slots:
    void initTestCase() {
        m_db = new Datenbank(this);
        QVERIFY(m_db->oeffne(":memory:"));
        m_kontoDienst = new KontoDienst(m_db, this);
        m_cache = new Zwischenspeicher(this);
    }

    void sollteKomplettenFlowDurchlaufen() {
        // 1. Konto anlegen
        auto id = m_kontoDienst->kontoAnlegen(
            "max@test.de", "Max", "imap.test.de", 993,
            "smtp.test.de", 587, "max", "geheim");
        QVERIFY(id > 0);

        // 2. IMAP mit FakeSocket
        ImapVerbindung imap;
        FakeSocket *fake = new FakeSocket(&imap);
        imap.setzeSocket(fake);

        PostfachDienst postfach(m_cache, &imap);
        QSignalSpy spyOrdner(&postfach, &PostfachDienst::ordnerGeladen);

        // 3. Verbinden
        postfach.verbinden("imap.test.de", 993, "max", "geheim");
        fake->simuliereEmpfang("* OK IMAP4rev1 Ready\r\n");
        fake->simuliereEmpfang("A0001 OK LOGIN\r\n");
        fake->simuliereEmpfang("* LIST () \"/\" \"INBOX\"\r\nA0002 OK\r\n");

        QCOMPARE(spyOrdner.count(), 1);
        QVERIFY(postfach.ordner().contains("INBOX"));
    }

private:
    Datenbank *m_db = nullptr;
    KontoDienst *m_kontoDienst = nullptr;
    Zwischenspeicher *m_cache = nullptr;
};

QTEST_MAIN(TestIntegration)
#include "tst_integration.moc"
```

**CMakeLists.txt:**
```cmake
mailadler_pruefung_hinzufuegen(tst_integration)
```

**Bauen + Test:**

```bash
cmake --build bau --parallel $(nproc)
ctest --test-dir bau --output-on-failure -R integration -V
```

Erwartet: 1 Test PASS.

**Commit:**

```bash
git add pruefungen/tst_integration.cpp pruefungen/CMakeLists.txt
git commit -m "prüfung: End-to-End-Integrationstest (DB→Dienst→IMAP-Fake)"
```

## Phase 2: Flatpak-Paket

### Aufgabe 2: Flatpak-Manifest fertigstellen + Bau testen

**Ziel:** `flatpak-builder` erzeugt ein lauffähiges Paket.

**Dateien:**
- Bearbeiten: `verpackung/flatpak/org.mailadler.MailAdler.yaml`
- Erstellen: `verpackung/flatpak/org.mailadler.MailAdler.desktop`
- Erstellen: `verpackung/flatpak/org.mailadler.MailAdler.metainfo.xml`

**Schritt 1: Desktop-Datei**

`verpackung/flatpak/org.mailadler.MailAdler.desktop`:
```ini
[Desktop Entry]
Name=MailAdler
Comment=Plattformübergreifender E-Mail-Client
Exec=mailadler
Icon=org.mailadler.MailAdler
Type=Application
Categories=Network;Email;
```

**Schritt 2: Metainfo**

`verpackung/flatpak/org.mailadler.MailAdler.metainfo.xml`:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<component type="desktop-application">
  <id>org.mailadler.MailAdler</id>
  <name>MailAdler</name>
  <summary>Plattformübergreifender E-Mail-Client</summary>
  <developer_name>MailAdler-Entwickler</developer_name>
  <launchable type="desktop-id">org.mailadler.MailAdler.desktop</launchable>
</component>
```

**Schritt 3: Manifest prüfen**

Das bestehende `org.mailadler.MailAdler.yaml` prüfen, ob alle Dateien referenziert sind.

**Schritt 4: Bauen (nur wenn flatpak-builder installiert)**

```bash
flatpak-builder --user --install build-dir verpackung/flatpak/org.mailadler.MailAdler.yaml
```

**Commit:**

```bash
git add verpackung/flatpak/
git commit -m "verpackung: Flatpak Desktop+Metainfo-Dateien"
```
